Usage:
\ffmv source destination\f7

Moves the file at \ffsource\f7 to \ffdestination\f7
If the file already exists at the destination, it is \f8deleted\f7

Functionally equivalent to running \ffcp source destination\f7
	and \ffrm source\f7 in succession.
