Usage:
\ffsave [filename]\f7

Save the cartridge in \fe/ram/cart\f7 to disk at the specified location.

If \fffilename\f7 doesn't contain a valid extension,
it will be given the .p64 extension automatically.
\f6Valid extensions:
\fa.p64 .p64.rom .p64.png

\f6If no location is specified:\f7
	The cartridge is saved to the current working cartridge location.
\f6If there is no current working cartridge location:\f7
	The cartridge is saved as \feuntitled.p64\f7"
